# Example Shiny app
---------------------
I simply write a shiny app to simulate the Dixon & Coles 1996 model :

```{r}
shiny::runGitHub('Dixon-Coles1996', 'englianhu', subdir='knitr')
shiny::runGitHub('Dixon-Coles1996', 'englianhu', subdir='shiny', display.mode='showcase')
```
